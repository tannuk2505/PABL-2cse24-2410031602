# Array Algorithms in Python

This repository contains solutions to five classic array problems:

1. Reverse an array in-place (`reverse_array.py`)
2. Find minimum and maximum in an array (`min_max_array.py`)
3. Find the k-th smallest element (`kth_smallest.py`)
4. Union of two arrays (`union_arrays.py`)
5. Find the largest element (`largest_element.py`)

## How to Run

Each file can be executed directly:

```bash
python filename.py
```

Replace `filename.py` with the script you want to run.

## Requirements
- Python 3.x
